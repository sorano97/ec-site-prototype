# morillion 女性向けECサイト

静的HTMLで構成された防犯用品紹介サイトです。

## フォルダ構成

- `content/`: コラムなどの原稿
- `docs/`: 調査資料とデザイン参考資料
- `images/`: 公開サイトで使用する画像
- `scripts/`: 商品画像取得や価格確認の運用スクリプト
- `source-assets/`: 公開サイトでは未使用の制作素材と旧画像

## ローカル起動

```sh
python3 -m http.server 8000 --bind 127.0.0.1
```

ブラウザで `http://127.0.0.1:8000/` を開きます。
